This style for Windows Vista/7 changes:
* Smaller Firefox Button.
* Smaller addon bar in bottom right corner (Show/hide with Ctrl+Shift+7).
* Aero Glass Download Manager.
* Aero Glass about:blank tab with Firefox Logo.
